## Design Context

### Users
Students (including the creator) who need to track tasks and deadlines with visual countdown timers. They want to feel urgency from ticking clocks and satisfaction from completing tasks.

### Brand Personality
Calm focus, satisfying completion, soft luxury — "creamy elegance meets productive urgency"

### Aesthetic Direction
- **Light theme**: Creamy light purples, soft lavender gradients, warm whites with purple tint
- **Dark theme**: Deep purples, dark backgrounds with purple undertones
- **Typography**: Huge, beautiful display fonts for timers and headings
- **Feel**: The ticking timer should create gentle urgency; the checkbox completion should feel deeply satisfying with smooth exit animations
- **Anti-references**: Corporate todo apps, sterile minimal tools, generic AI gradient pages
- **Key moments**: Hero countdown timer at top, satisfying circle checkbox click, tasks animating away on completion

### Design Principles
1. **Time is visual** — timers should feel alive and prominent, not hidden
2. **Completion is reward** — clicking done should feel as good as finishing the task
3. **Soft but focused** — creamy gradients create calm; ticking seconds create focus
4. **Effortless flow** — adding, sorting, and completing tasks should feel instant
5. **Student-friendly** — zero friction, local storage, no accounts needed
